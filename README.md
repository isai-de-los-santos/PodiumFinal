# Podium App 🏆

Podium es una aplicación móvil multiplataforma que optimiza la entrega, visualización y evaluación de proyectos académicos. Permite a los estudiantes subir sus trabajos junto con sus evidencias en video, y otorga a los profesores las herramientas necesarias para visualizarlos y otorgar calificaciones sin fricciones.

## 🚀 Características Principales
- **Sistema Multiusuario:** Experiencias personalizadas para Estudiantes, Profesores y Administradores.
- **Gestión de Proyectos Estudiantiles:** Creación de proyectos, adición de colaboradores y descripción detallada.
- **Subida de Evidencias (Videos):** Carga fluida de clips de introducción, desarrollo y evidencia extra.
- **Panel de Evaluación para Profesores:** Visualizador de videos optimizado, vista de metadatos del proyecto y sistema de puntuación.
- **Sincronización en Tiempo Real:** Backend gestionado ágilmente a través de Supabase.

## 🛠️ Tecnologías Utilizadas
- **Framework:** [React Native](https://reactnative.dev/) v0.81 + [Expo](https://expo.dev/) v54
- **Navegación:** Expo Router
- **Estilos:** NativeWind (Tailwind CSS)
- **Backend & Auth:** Supabase
- **Lenguaje:** TypeScript
